My Slide Image Downloader

[App Description]

My Slide Image Downloader is a Chrome Extension designed to help users batch download images and graphics from their active Google Slides presentations into a single ZIP file.

Privacy Policy

Effective Date: May 27, 2026

This Privacy Policy governs the use of the Chrome Extension "My Slide Image Downloader".

1. Zero Data Collection Policy

We take your privacy very seriously. The Extension operates under a strict zero-data collection policy. We DO NOT collect, store, transmit, or share any personal information, account credentials, or presentation contents. All data processing, image extraction, and ZIP compression occur strictly within your local browser environment.

2. Explanation of Permissions Used

  • presentations.readonly: Used exclusively to read the image elements within the user's active Google Slides via the official API for the batch download feature.
  • identity: Used to obtain a temporary OAuth token required to authenticate requests to the Google Slides API locally.
  • downloads: Used to trigger Chrome's native download manager to save the extracted images or generated ZIP file to your computer.
  • activeTab / Host Permissions: Used to access the active Google Slides tab to detect right-click interactions on images and fetch their original source URLs.
